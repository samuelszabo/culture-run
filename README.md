# Culture Run

Behačka pre prehliadač (počítač aj mobil), v ktorej hráč spoznáva krajiny sveta — vyberie si krajinu podľa vlajky, zaujímavé miesto z nej a beží cez neho, vyhýba sa prekážkam a zbiera miestne jedlo. Za perfektný beh (5 hviezdičiek) získava kozmetické odmeny pre svoju postavičku.
